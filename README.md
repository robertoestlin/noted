# Noted

**Your short-term text workspace.**

## About

Noted is a lightweight tab-based text editor for anything you want to keep for up to about three months.  
It is designed for short- to medium-term information such as tasks, ideas, meeting notes, project context, reference snippets, and todos.  
The app auto-saves all open tabs into timestamped backups, frequently to disk and optionally to a cloud-backed directory if wanted, so your working notes are preserved without manual save steps.  
Notes can be quickly organized with tab rename, close, drag-to-reorder, and per-line highlight support for important details.  
Noted focuses on fast capture and retrieval of active notes, not long-term archival or heavy document management.

## Install

You can install Noted in one of two ways:

1. Build from source code
   - Install the .NET 9 SDK on Windows.
   - Clone this repository and open a terminal in the project folder.
   - Run `dotnet build` to compile, then run `dotnet run` to start the app.

2. Use a prebuilt binary
   - Download the latest Windows `.exe` build provided with the project release.
   - Launch `Noted.exe`.
